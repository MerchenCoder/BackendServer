## 프로젝트 소개
<메르헨 코더 : 코딩으로 동화 세계를 구하라!>는 코딩 학습에 흥미가 있는 초등학생을 대상으로 한 코딩 학습 모바일 게임입니다. 유니티와 C# 스크립트를 활용하여 직접 개발한 비주얼 프로그래밍 도구로 프로그래밍 학습을 진행합니다. 게임의 서버는 Node.js로 구현되었으며, 다음의 주요 기능을 제공합니다.

### 주요 기능
- 회원가입
- 로그인
- 계정 삭제
- 게임 데이터 저장
- 게임 데이터 불러오기
- S3에 저장된 정적 파일 읽기 & 클라이언트로 전송하기
